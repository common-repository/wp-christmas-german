== Description ==

WP-Christmas german l&auml;sst Schneeflocken und [Weihnachtsgeschenke](http://www.weihnachtsgeschenk.com/) &uuml;ber Deinen Blog rieseln. 
Das Plugin hat eine eigene Konfigurationsseite mit der einige Einstellungen vorgenommen werden k&ouml;nnen (Anzahl Schneeflocken, Geschwindigkeit der Schneeflocken usw.).

== Installation ==

1. Entpacke `wp-christmas-german.1.0.zip`.
2. Lade den `wp-christmas-german`-Ordner in dein Verzeichnis `/wp-content/plugins` hoch.
3. Aktiviere das Plugin im 'Plugins'-Men&uuml; der Wordpress-Administration.
4. Konfiguriere das Plugin unter seinen Einstellungen.
